发表于 2018-08-01 更新于 2021-07-21 分类于 Network

上篇文章<<D-Bus实例介绍>>中简要介绍了D-Bus的基本概念，其中提到systemd、NetworkManager等系统服务导出了D-Bus API供其他程序来调用。本文通过示例来说明这些API的查找与调用。
